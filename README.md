### 🎲 Rodando Aplicativo web

```bash
# Clone este repositório
$ git clone git@github.com:ArielKollross/Github-Projects-Repository.git

# Acesse a pasta do projeto no terminal/cmd
$ cd web

# Instale as dependências
$ yarn

#ou com npm
$ npm install

# Execute a aplicação 
$ yarn start
# ou com npm
$ npm start
```
